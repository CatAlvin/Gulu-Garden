## Version 0.6.0 — 单存档系统

日期：2026-05-29

### 已完成

- 创建 `src/systems/save_system.py`
- 添加 `SaveSystem` 存档系统
- 使用 JSON 文件保存游戏进度
- 默认存档路径为 `saves/save_1.json`
- 退出游戏时自动保存当前进度
- 支持按 `S` 手动保存游戏
- 启动游戏时自动读取已有存档
- 如果存档不存在，则自动开始新游戏

- 保存玩家金币数量
- 保存背包中的星泡萝卜种子数量
- 保存 8 块土地的状态
- 保存土地是否解锁
- 保存土地上的作物 ID
- 保存作物种植时间
- 保存作物当前成长阶段
- 读取存档后可以恢复金币、种子、土地和作物状态
- 真实存档文件 `saves/save_1.json` 不提交到 GitHub

### 当前存档内容

当前 `save_1.json` 主要保存：

- `save_version`
- `game_version`
- `slot_id`
- `created_at`
- `last_saved_at`
- `player.coins`
- `inventory.seeds`
- `plots`
- 每块土地上的 `crop_id`
- 每块土地上的 `planted_at`
- 每块土地上的 `current_stage`

### 暂未实现

- 存档损坏后的 UI 提示
- 自动备份存档
- 离线成长总结提示
- 系统时间异常检测

### 当前结论

Version 0.6 的核心目标已经完成：游戏可以把当前进度保存到本地 JSON 文件，并在下一次启动时恢复进度。玩家关闭游戏后，金币、种子、土地和作物状态不会丢失。
