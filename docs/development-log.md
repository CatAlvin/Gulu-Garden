## Version 1.0.2 — 昼夜背景贴图与场景调色系统

日期：2026-06-01

### 版本定位

Version 1.0.2 是《咕噜小菜园》（Gulu Garden）的第二轮视觉系统升级。

这一版本的重点不是新增玩法，而是完善昼夜视觉表现：让游戏根据现实时间或开发预览键切换清晨、白日、傍晚和午夜背景，并让土地、作物等场景素材根据当前昼夜阶段进行简易调色。

### 已完成

- 接入四个昼夜背景图
  - `farm_morning.png`
  - `farm_daytime.png`
  - `farm_evening.png`
  - `farm_midnight.png`
- 新增 `DAY_PHASE_BACKGROUND_IMAGES` 配置
- 根据当前昼夜阶段绘制对应背景图
- 如果当前阶段背景缺失，自动回退到 daytime 背景
- 如果 daytime 背景也缺失，继续回退到纯色背景
- 新增 `SCENE_TINT_SETTINGS` 场景调色配置
- 新增 `src/utils/image_tint.py`
- 扩展 `AssetLoader`，支持按昼夜阶段缓存调色后的图片
- 土地贴图支持昼夜调色
- 作物贴图支持昼夜调色
- 调色结果会缓存，不会每帧重新计算
- 修复调色时污染透明背景的问题
- 调色逻辑只修改 RGB，不修改 Alpha
- UI、文字、按钮暂时不参与场景调色，保证可读性
- 添加昼夜预览调试键
  - `1`：Morning
  - `2`：Daytime
  - `3`：Evening
  - `4`：Midnight
  - `0`：恢复真实时间
- HUD 在预览模式下显示 `(Preview)`
- 存档时不会保存调试预览阶段，而是保存真实时间阶段
